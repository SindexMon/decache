Internet stuff is important.

...and you might have some of it.

When you visit a website, all the files it has to load in (like images, videos, games, etc.) are temporarily stored in what's called the cache. As you continue to browse the internet, these files are eventually deleted to save space.

Sometimes, they aren't. That's the key.

Decache will automatically find these files and compare them against hundreds of pieces of lost media.

How to use

Upon extracting the program and running start_decache.bat, you'll be asked to select a computer to scan. The computer you're running Decache on will show up with "C:" in the name.

If you're unsure about what exactly a "computer" means in this case, what you need to look for is a Windows hard drive. These will contain a folder called Users or Documents and Settings, though user-specific backups (i.e. those with a Desktop folder) are also supported.

For more information, see the FAQ.
